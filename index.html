Ceramic

A cross-platform & open-source 2D framework

Ceramic is a cross-platform framework that lets you create 2D apps and games written in the Haxe programming language.

Cross-platform, powered by Haxe

Thanks to the Haxe programming language,
you can write your code once and transpile it to C++, JS or C#.

Ceramic takes advantage of that feature to export your app natively to
Windows, Mac, Linux, iOS, Android, HTML5/WebGL and even Unity projects!
